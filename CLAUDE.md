# CLAUDE.md

Compatibility wrapper for AI tooling that expects repo-local instructions in
`CLAUDE.md`.

The maintained source of truth for automation and AI contributor guidance is
[`docs/AI_RULES.md`](./docs/AI_RULES.md).

If you are updating AI guidance, edit [`docs/AI_RULES.md`](./docs/AI_RULES.md)
instead of this file.
